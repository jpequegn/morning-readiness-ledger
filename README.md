# Morning Readiness Ledger

A local-first morning check-in for planning endurance training and work capacity from subjective signals before reviewing wearable context. The app stores data in the current browser only.

## Capabilities

- Six-choice daily flow that records nine subjective readiness dimensions and an optional note.
- Transparent `Build`, `Maintain`, `Protect`, and `Recover` capacity bands with illness and injury safety overrides.
- Wearable context deliberately withheld until the subjective check-in is complete.
- Editable date-keyed local entries, a seven-record review, raw-answer drill-down, JSON export, and local reset.

## Development

```bash
npm install
npm run dev
```

Run the quality checks with `npm run lint`, `npm test`, and `npm run build`.
